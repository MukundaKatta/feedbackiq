# FeedbackIQ

> AI-Powered Customer Feedback Intelligence

## Overview

FeedbackIQ transforms raw customer feedback into actionable intelligence. It aggregates reviews, support tickets, surveys, and social mentions, then uses NLP to extract sentiment, themes, and product insights at scale.

## Key Features

- **Multi-Source Ingestion** — Reviews, tickets, surveys, social media, app store ratings
- **Sentiment Analysis** — Fine-grained sentiment beyond positive/negative
- **Theme Extraction** — Automatically clusters feedback into product themes
- **Trend Detection** — Spot emerging issues before they become crises
- **Competitive Analysis** — Compare your feedback against competitors
- **Priority Scoring** — AI ranks feedback by business impact
- **Executive Dashboards** — Auto-generated insight reports for stakeholders

## Tech Stack

- **Backend:** Python, FastAPI
- **NLP:** Claude API, spaCy, sentence-transformers
- **Database:** PostgreSQL, Elasticsearch
- **Frontend:** React, Recharts
- **Deployment:** Docker, GCP

## Getting Started

```bash
git clone https://github.com/MukundaKatta/feedbackiq.git
cd feedbackiq
pip install -e .
feedbackiq analyze --source reviews.csv
```
